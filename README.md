# Porcelain Mask
**Library for my mods.**
<hr>



**Enables doing things such as:**
* Changing player reach with specific items.
* Having different models for a single item. (GUI, Main-Hand, Offhand, etc...)
* Enabling *"Sweep"* attacks on specific items.
